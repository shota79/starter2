Checkbox

기본 체크박스는 HTML, CSS로만 작동합니다.

- CODE / 기본

HTML

    <label class="pualugin-checkbox">
      <input type="checkbox" class="pualugin-checkbox__input">
      <i class="pualugin-checkbox__icon fas fa-check"></i>
      <span class="pualugin-checkbox__text">동의합니다.</span>
    </label>

- RESULT

전체선택 기능이 필요한 경우 아래와 같이 작성합니다.

HTML

  <div data-element="checkbox">
    <label class="pualugin-checkbox">
      <!-- 일부 체크박스를 비활성화 해야 하는 경우 disabled 속성을 추가합니다. -->
      <input type="checkbox" class="pualugin-checkbox__input" data-element="checkbox__input" disabled>
      <i class="pualugin-checkbox__icon fas fa-check"></i>
      <span class="pualugin-checkbox__text">동의합니다.</span>
    </label>
    <label class="pualugin-checkbox">
      <input type="checkbox" class="pualugin-checkbox__input" data-element="checkbox__input">
      <i class="pualugin-checkbox__icon fas fa-check"></i>
      <span class="pualugin-checkbox__text">동의합니다.</span>
    </label>
    <label class="pualugin-checkbox">
      <input type="checkbox" class="pualugin-checkbox__input" data-element="checkbox__input">
      <i class="pualugin-checkbox__icon fas fa-check"></i>
      <span class="pualugin-checkbox__text">동의합니다.</span>
    </label>
    <label class="pualugin-checkbox">
      <input type="checkbox" class="pualugin-checkbox__input" data-element="checkbox__all">
      <i class="pualugin-checkbox__icon fas fa-check"></i>
      <span class="pualugin-checkbox__text">전체선택</span>
    </label>
  </div>
