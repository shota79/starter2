Tooltip & Popover

Tooltip

- CODE: 기본 및 옵션 추가

HTML

  <div class="pualugin-tooltip" data-element="tooltip">
    <button class="pualugin__button" type="button" data-element="tooltip__button">Tooltip - right</button>
    <span class="pualugin-tooltip__panel" data-element="tooltip__panel">
        to right.
    </span>
  </div>

  <div class="pualugin-tooltip" data-element="tooltip" data-options='{ "position": "left" }'>
    <button class="pualugin__button" type="button" data-element="tooltip__button">Tooltip - left</button>
    <span class="pualugin-tooltip__panel" data-element="tooltip__panel">
        to left.
    </span>
  </div>

  <div class="pualugin-tooltip" data-element="tooltip" data-options='{ "position": "top" }'>
    <button class="pualugin__button" type="button" data-element="tooltip__button">Tooltip - top</button>
    <span class="pualugin-tooltip__panel" data-element="tooltip__panel">
        to top.
    </span>
  </div>

  <div class="pualugin-tooltip" data-element="tooltip" data-options='{ "position": "bottom" }'>
      <button class="pualugin__button" type="button" data-element="tooltip__button">Tooltip - bottom</button>
      <span class="pualugin-tooltip__panel" data-element="tooltip__panel">
          to bottom.
      </span>
  </div>

- RESULT

to right.
to left.
to top.
to bottom.

Popover

- CODE: 기본 및 옵션 추가

HTML

  <div class="pualugin-tooltip" data-element="tooltip" data-options='{"mode": "popover"}'>
    <button class="pualugin__button" type="button" data-element="tooltip__button">Tooltip - right</button>
    <span class="pualugin-tooltip__panel" data-element="tooltip__panel">
        to right.
    </span>
  </div>

  <div class="pualugin-tooltip" data-element="tooltip" data-options='{ "position": "left" }'>
    <button class="pualugin__button" type="button" data-element="tooltip__button">Tooltip - left</button>
    <span class="pualugin-tooltip__panel" data-element="tooltip__panel">
        to left.
    </span>
  </div>

  <div class="pualugin-tooltip" data-element="tooltip" data-options='{ "position": "top" }'>
    <button class="pualugin__button" type="button" data-element="tooltip__button">Tooltip - top</button>
    <span class="pualugin-tooltip__panel" data-element="tooltip__panel">
        to top.
    </span>
  </div>

  <div class="pualugin-tooltip" data-element="tooltip" data-options='{ "position": "bottom" }'>
      <button class="pualugin__button" type="button" data-element="tooltip__button">Tooltip - bottom</button>
      <span class="pualugin-tooltip__panel" data-element="tooltip__panel">
          to bottom.
      </span>
  </div>

- RESULT

to right.
to left.
to top.
to bottom.
Options
  • mode
    • type: String
    • default: tooltip
    • description: tooltip, popover 중 선택
  • position
    • type: String
    • default: right
    • description: right, left, top, bottom 중 선택
  • indent
    • type: Number
    • default: 10
    • description: Button 과 Panel의 간격
  • activeClassName
    • type: String
    • default: is-active
    • description: 토글 활성화 클래스

- Method

JAVASCRIPT 

  var yourVariable = $('.your-element').data('plugin_tooltip');

  yourVariable.show();

  yourVariable.hide();

  yourVariable.destroy();

  yourVariable.reInit();
  • show
    • arguments: none
    • description: 패널 열기
  • hide
    • arguments: none
    • description: 패널 닫기
  • destroy
    • arguments: none
    • description: 플러그인 제거
  • reInit
    • arguments: none
    • description: 플러그인 제거 후 생성

- 모든 플러그인은 기본적으로 페이지 렌더링시 Initialized 됩니다. 개별적(비동기 마크업 구현) Initialize 필요한 경우 아래와 같이 진행합니다.

JAVASCRIPT 

  $('.your-element').tooltip(); // Initialize
