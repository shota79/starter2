Options
  • mode
    • type: String
    • default: static
    • description: static, slide, fade 중 선택
  • speed
    • type: Number
    • default: 300
  • easing
    • type: String
    • default: swing
  • event
    • type: String
    • default: click
    • description: click, focusin 중 선택
  • initIndex
    • type: Number
    • default: 0
    • description: 원하는 탭 순서 노출
  • autoScroll
    • type: Boolean
    • default: false
    • description: 탭 클릭 시 해당 탭의 상단까지 자동으로 스크롤 여부
  • activeClassName
    • type: String
    • default: is-active
    • description: 탭 활성화 클래스
  • disabledClassName
    • type: String
    • default: is-disabled
    • description: 탭 비활성화 클래스
  • selectedText
    • type: String
    • default: Selected
    • description: title 속성값

- Event

JAVASCRIPT 

  var yourVariable = $('.your-element');

  yourVariable.on('beforeChange', function( event, plugin, anchor, panel ){
      /* your code ... */
  })

  yourVariable.on('afterChange', function( event, plugin, anchor, panel ){
      /* your code ... */
  })
  • beforeChange
    • Arguments: event, plugin, anchor, panel
    • description: 패널 오픈 전 실행될 함수
  • afterChange
    • Arguments: event, plugin, anchor, panel
    • description: 패널 오픈 후 실행될 함수

- Method

JAVASCRIPT 

  var yourVariable = $('.your-element').data('plugin_tab');

  yourVariable.open(1); //OR
  yourVariable.open(2, true); //OR
  yourVariable.open(0, false); //OR

  yourVariable.destroy();

  yourVariable.reInit();
  • open
    • Arguments: index, autoScroll(Boolean): 선택된 탭의 상단으로 스크롤 이동 여부
    • description: 원하는 탭 오픈
  • destroy
    • Arguments: none
    • description: 플러그인 제거
  • reInit
    • Arguments: none
    • description: 플러그인 제거 후 생성

- 모든 플러그인은 기본적으로 페이지 렌더링시 Initialized 됩니다. 개별적(비동기 마크업 구현) Initialize 필요한 경우 아래와 같이 진행합니다.

JAVASCRIPT 

  $('.your-element').tab(); // Initialize
