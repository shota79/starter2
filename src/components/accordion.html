Options
  • mode
    • type: String
    • default: static 중 선택
    • description: static, slide, fade 중 선택
  • speed
    • type: Number
    • default: 300
  • easing
    • type: String
    • default: swing
  • isInitActive
    • type: Boolean
    • default: true
    • description: 초기화시 패널 오픈여부
  • initIndex
    • type: Number
    • default: 0
    • description: 초기화시 오픈될 패널의 index
  • autoFold
    • type: Boolean
    • default: true
    • description: 탭 클릭 시 기존에 열려있던 탭 닫기
  • autoScroll
    • type: Boolean
    • default: false
    • description: 선택한 탭의 위치로 자동 스크롤
  • activeClassName
    • type: String
    • default: is-active
    • description: 아코디언 활성화 클래스

- Event

JAVASCRIPT 

  var yourVariable = $('.your-element');

  yourVariable.on('beforeChange', function( event, plugin, anchor, panel ){
      /* your code ... */
  })
  
  yourVariable.on('afterChange', function( event, plugin, anchor, panel ){
      /* your code ... */
  })
  • beforeChange
    • arguments: event, plugin, anchor
    • description: 패널 오픈 전 실행될 함수
  • afterChange
    • arguments: event, plugin, anchor
    • description: 패널 오픈 후 실행될 함수

- Method

JAVASCRIPT 

  var yourVariable = $('.your-element').data('plugin_accordion');

  yourVariable.go(1); //OR
  yourVariable.go(2, true); //OR
  yourVariable.go(0, false); //OR

  yourVariable.destroy();

  yourVariable.reInit();
  • go
    • arguments: index, withScroll(Boolean): 선택된 탭의 상단으로 스크롤 이동 여부
    • description: 패널 열기
  • destroy
    • arguments: none
    • description: 플러그인 제거
  • reInit
    • arguments: none
    • description: 플러그인 제거 후 생성

- 모든 플러그인은 기본적으로 페이지 렌더링시 Initialized 됩니다. 개별적(비동기 마크업 구현) Initialize 필요한 경우 아래와 같이 진행합니다.

JAVASCRIPT 

  $('.your-element').accordion(); // Initialize
